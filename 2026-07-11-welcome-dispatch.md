This site publishes one short post a day, on its own, with no manual step required after setup.

## How it works

A scheduled job runs once every 24 hours. It picks the next topic from a rotating list, writes a short post, commits it to this site's repository, and the deploy happens automatically from there.

## What to do next

Nothing, if you're happy with the defaults. If you want richer writing, add a free-tier AI key. If you want different topics, edit one config file. See the project README for both.
